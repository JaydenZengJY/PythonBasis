# PythonBasis

作为一个新手小白，从今天开始，我将正式开始学习 **MuJoCo 和 AI Agent**，并在此仓库中同步更新，记录自己的学习情况。

## 目标
- 夯实 Python 基础，为后续机器人仿真与强化学习开发铺路
- 成长为能独立完成项目、具备工程能力的开发者，冲击大厂相关岗位

---

## 具体学习安排

### 1. 第 1–2 周：函数（`def`）彻底弄懂
- 函数的定义与调用
- 参数详解：位置参数、默认参数、关键字参数
- 返回值：单值返回、多值返回
- 函数嵌套与作用域
- `lambda` 匿名函数的简单用法

---

### 2. 第 3–4 周：类与面向对象（`class`）
- `class` 的定义与实例化
- 构造方法 `__init__` 与 `self` 的含义
- 实例方法、属性访问
- 类的继承与方法重写
- `__str__` 方法：自定义对象的字符串表示

---

### 3. 第 5 周：异常处理与模块化
- 异常处理：`try/except/finally`
- 模块化编程：
  - `import` 导入自定义模块
  - `if __name__ == "__main__":` 的用法
- 常用标准库入门：`os` / `sys` / `random`

---

### 4. 第 6 周：综合练手项目
**项目目标：用面向对象的方式写一个“机器人”对象**

- 机器人对象属性：
  - 位置（x, y）
  - 速度、方向等（可选）
- 机器人对象方法：
  - `move()`：更新位置
  - `check_collision()`：碰壁检测
- 可视化：
  - 使用 `matplotlib` 画出机器人的运动轨迹
